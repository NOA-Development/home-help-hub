## Features Demonstrated

### Leaflet Map Integration
- ✅ Free and open-source alternative to Google Maps
- ✅ OpenStreetMap tiles
- ✅ Custom markers for user and specialist
- ✅ Interactive location selection on address input
- ✅ Animated specialist movement towards user
- ✅ Distance-based ETA calculation

### Call Dialog
- ✅ Beautiful gradient background with animations
- ✅ Live call duration counter
- ✅ Mute/unmute microphone control
- ✅ Speaker on/off control
- ✅ Visual audio indicator
- ✅ Smooth animations matching app style

### Message Dialog
- ✅ Real-time chat interface
- ✅ Auto-replies from specialist (mocked)
- ✅ Typing indicator
- ✅ Message timestamps
- ✅ Send message on Enter key
- ✅ Smooth animations

### Synchronized ETA
- ✅ ETA countdown synchronized with specialist movement
- ✅ Real-time distance calculation
- ✅ Smooth specialist marker animation (0.8% movement per update)
- ✅ Dynamic ETA updates based on actual distance
